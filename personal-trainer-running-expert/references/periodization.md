# Running periodization

How to build a phased plan from where the athlete is now to a goal on a date.

## Principles
- **Reverse-plan from the goal date.** Place the event, then work backwards through taper,
  peak, build, and base to today.
- **General → specific.** Early work is broad (aerobic base, strength, economy); later work
  looks like the goal (race-pace specificity).
- **Develop the limiter.** Spend the build on whatever most constrains the goal (endurance
  for the marathon, speed/threshold for the 5k).

## Phase templates
- **Base** — aerobic volume; mostly easy running plus strides and hills; introduce light
  threshold late. Build the engine and durability.
- **Build** — add threshold and VO2max work and longer quality; volume holds or rises
  slightly; sessions start to resemble the race.
- **Peak / sharpen** — race-pace specificity; trim a little volume to absorb the quality.
- **Taper** — cut volume ~30–50% over 1–3 weeks (longer for the marathon), keep some
  intensity to stay sharp; arrive fresh.
- **Recovery / transition** — after the goal: easy, unstructured, reset before the next block.

## By goal
- **5k / 10k** — threshold + VO2max emphasis; race-pace reps; modest volume, high quality.
- **Half-marathon** — threshold-centric; long runs with segments at half/marathon pace.
- **Marathon** — large aerobic base; long-run progression; marathon-pace blocks; rehearse
  fueling; the longest taper.
- **General fitness** — balanced easy running + 1–2 quality sessions; no formal taper needed;
  keep it sustainable and consistent.
- **Hyrox run portion** — repeated ~1 km efforts run *under fatigue* between stations.
  Emphasize threshold and the ability to hold moderate pace when tired ("compromised
  running" — run off tired legs). The strength/functional side is owned by a separate
  sport-expert skill; here, cover only the running.

## Laying phases against the calendar
- Count the weeks to the goal; allocate them across phases (a common split for a multi-month
  build is roughly base 40% / build 35% / peak 15% / taper 10% — adjust to the athlete and
  the time available).
- Schedule a benchmark (time-trial or tune-up) every ~3–4 weeks to recalibrate paces and
  confirm progress.

## Adjusting from data
If adaptation stalls or fatigue accumulates across weekly reviews, don't push harder —
extend the base, insert an extra recovery week, or shorten the build. Periodization is a
plan, not a contract; the data and the athlete revise it.
